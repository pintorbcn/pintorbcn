About This Gig

If you're looking to have a caricature drawn from a picture you provide, you're in the right place.

If you want a hand-drawn fully rendered caricature, black and white or color, please message me because these take more time to complete.


Things to keep in mind:

1. The caricature will resemble the picture you provide; choose your photo wisely.

2. I need to be able to zoom in on the pictures and have them still look clear, not fuzzy.

3. Any options not included in a package can be added for an additional charge.

4. I'll message you if I have any questions.

5. Message me if you have any questions and/or special requests before you order.
